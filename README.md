# Tools

### 1. zip

`node zip.js -P filesfolder -E password -O outputfile`

* -P: 指定打包文件夹
* -E: 打包密码
* -O: 输出文件夹

### 2. unzip

`node unzip.js -P outputfile -E password -O outputfolder`

* -P: 指定解压文件
* -E: 解压密码
* -O: 解压输出目录

### 3. m3u8download

`node m3u8download playlist.m3u8`

会下载视频到项目 stream目录
